So What is an API?

First and foremost, an API is an Application Programming Interface. It sounds really complicated, but it is actually just a way for two pieces of software to interact in a way such that one end provides the other with instructions while the other, given these instructions, will return information and data back to the other. Just imagine ordering a pizza and asking for a large pie with pineapples and cheesy crust. Assuming the pizzeria does their job correctly, you will receive the pizza you ordered! See? no sweat.
